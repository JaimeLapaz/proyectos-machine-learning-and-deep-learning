### 💡 Description

This repository includes a variety of machine learning projects covering various domains and methodologies, from classification and regression to clustering and neural networks. Each project is accompanied by source code, datasets (when applicable), and a Jupyter Notebook with a thorough explanation and visualization of the results. These projects showcase different stages in the data science workflow, from data preprocessing and feature engineering to model training and evaluation.

### 📈 Machine Learning Projects

#### 🔹 Supervised Learning

- **Project 1: Cardiovascular Disease Classification ML**  
  - **Description:** This project predicts whether an individual has cardiovascular disease based on health-related features such as age, blood pressure, cholesterol levels, and more. By applying multiple supervised learning algorithms like Decision Trees, Logistic Regression, and SVM, we aim to identify the most accurate model for this classification task. This project includes feature selection, cross-validation, and performance evaluation to ensure robust results.  
  - **Technologies:** Python, scikit-learn, pandas, matplotlib, seaborn  
  - **[View Project](./Proyectos_Machine_Learning/Supervised_Learning/01_Clasificacion_Enfermedades_Cardiovasculares)**

- **Project 2: Stress Level Prediction**  
  - **Description:** This regression project predicts the stress level of employees using a dataset containing various job-related factors such as work hours, job role, and work environment conditions. Different regression models (e.g., Linear Regression, Random Forest, and XGBoost) are used to determine the features most associated with stress. The project focuses on data preprocessing, model optimization, and visualization of the results, offering insights into stress factors in the workplace.  
  - **Technologies:** Python, scikit-learn, pandas, matplotlib, seaborn  
  - **[View Project](./Proyectos_Machine_Learning/Supervised_Learning/02_Prediccion_Nivel_Estres)**

#### 🔹 Unsupervised Learning

- **Project 1: Cardiovascular Disease Clusters**  
  - **Description:** This project aims to uncover hidden patterns in a cardiovascular disease dataset using unsupervised learning techniques. By applying clustering methods like K-Means and Hierarchical Clustering, we explore possible subgroupings within patients to investigate if there are distinct clusters associated with specific health profiles. This project is especially useful for identifying potential subcategories of cardiovascular risk that might not be evident from the raw labels.  
  - **Technologies:** Python, Pandas, Seaborn, Matplotlib, Scikit-learn  
  - **[View Project](./Proyectos_Machine_Learning/Unsupervised_Learning/01_Clusteres_enfermedades_cardiovasculares)**

- **Project 2: Stress Level Influences**  
  - **Description:** In this analysis project, we aim to identify the primary factors that influence stress levels among employees through clustering and exploratory data analysis. By clustering employees based on shared characteristics and performing Principal Component Analysis (PCA), this project sheds light on the work conditions or personal factors that most affect stress. This approach can help in developing targeted interventions to reduce stress in the workplace.  
  - **Technologies:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
  - **[View Project](./Proyectos_Machine_Learning/Unsupervised_Learning/02_Influencias_nivel_estres)**

### 🤖 Deep Learning Projects

- **Project 1: Cardiovascular Disease Classification DL**  
  - **Description:** This project builds a feed-forward neural network to predict cardiovascular diseases based on multiple health metrics. The neural network model is trained on structured health data and tuned for optimal accuracy using regularization techniques and hyperparameter tuning. Additionally, SHAP values are used to interpret the model's decisions, revealing which features contribute most to the model’s predictions.  
  - **Technologies:** TensorFlow, Keras, Scikit-Learn, SHAP  
  - **[View Project](./Proyectos_Deep_Learning/01_Clasificacion_Enfermedades_Cardiovasculares)**

- **Project 2: Image Classification**  
  - **Description:** This project implements a convolutional neural network (CNN) for classifying handwritten digits from the MNIST dataset. Regularization techniques such as Dropout and EarlyStopping are applied to prevent overfitting, while visualization tools like Confusion Matrix and Learning Curves are used to evaluate the model’s performance. This project demonstrates basic deep learning techniques for image classification.  
  - **Technologies:** TensorFlow, Keras, Matplotlib, Seaborn  
  - **[View Project](./Proyectos_Deep_Learning/02_Clasificacion_Imagenes)**

- **Project 3: Wine Quality Prediction**  
  - **Description:** Using a dataset containing physicochemical properties of white wines, this project develops a deep learning regression model to predict wine quality scores. Feature engineering and normalization techniques are used to preprocess the data, and a neural network model is trained to produce accurate predictions. The project provides insights into the characteristics that most strongly influence wine quality.  
  - **Technologies:** PyTorch, Numpy, Scikit-learn  
  - **[View Project](./Proyectos_Deep_Learning/03_Prediccion_Calidad_Vino)**

- **Project 4: Movie Recommender Using Word Embeddings**  
  - **Description:** This project utilizes Word2Vec embeddings to develop a recommendation system that suggests movies based on similarity in their synopses. Natural Language Processing (NLP) techniques are used to process text data, and vectorized synopses allow for efficient similarity calculations. This project offers insights into content-based recommendation systems in the entertainment domain.  
  - **Technologies:** spaCy, Gensim, Scikit-learn  
  - **[View Project](./Proyectos_Deep_Learning/04_Recomendador_peliculas)**

- **Project 5: Sentiment Analysis of Movie Reviews**  
  - **Description:** This project uses Long Short-Term Memory (LSTM) networks to analyze the sentiment of IMDb movie reviews. By employing word embeddings and sentiment classification techniques, the model categorizes reviews as positive or negative, helping to understand public perception of movies. The project includes data preprocessing, tokenization, and visualization of sentiment trends in the reviews.  
  - **Technologies:** TensorFlow/Keras, NLTK, Gensim  
  - **[View Project](./Proyectos_Deep_Learning/05_Analisis_Sentimientos_Reseñas_Peliculas)**

### 📈 Proyectos Machine Learning

#### 🔹 Aprendizaje Supervisado

- **Proyecto 1: Clasificación de Enfermedades Cardiovasculares ML**  
  - **Descripción:** Este proyecto predice si una persona padece enfermedad cardiovascular en función de características de salud como la edad, presión arterial, niveles de colesterol, entre otros. A través de algoritmos de aprendizaje supervisado como Árboles de Decisión, Regresión Logística y SVM, buscamos identificar el modelo más preciso para esta tarea. Incluye selección de características, validación cruzada y evaluación de rendimiento para asegurar resultados robustos.  
  - **Tecnologías:** Python, scikit-learn, pandas, matplotlib, seaborn  
  - **[Ver Proyecto](./Proyectos_Machine_Learning/Supervised_Learning/01_Clasificacion_Enfermedades_Cardiovasculares)**

- **Proyecto 2: Predicción del Nivel de Estrés**  
  - **Descripción:** Este proyecto de regresión predice el nivel de estrés de empleados mediante un conjunto de datos con factores laborales como horas de trabajo, tipo de puesto y condiciones del entorno. Modelos de regresión como Regresión Lineal, Random Forest y XGBoost son aplicados para identificar los factores asociados al estrés. Se enfoca en la preprocesamiento de datos, optimización del modelo y visualización de resultados para brindar información sobre los factores de estrés en el entorno laboral.  
  - **Tecnologías:** Python, scikit-learn, pandas, matplotlib, seaborn  
  - **[Ver Proyecto](./Proyectos_Machine_Learning/Supervised_Learning/02_Prediccion_Nivel_Estres)**

#### 🔹 Aprendizaje No Supervisado

- **Proyecto 1: Clústeres de Enfermedades Cardiovasculares**  
  - **Descripción:** Este proyecto busca patrones ocultos en un conjunto de datos de enfermedades cardiovasculares mediante técnicas de aprendizaje no supervisado. Con métodos de clustering como K-Means y Clustering Jerárquico, exploramos posibles subgrupos entre pacientes para investigar si existen clústeres asociados con perfiles de salud específicos, lo cual podría ayudar a identificar subcategorías de riesgo cardiovascular.  
  - **Tecnologías:** Python, Pandas, Seaborn, Matplotlib, Scikit-learn  
  - **[Ver Proyecto](./Proyectos_Machine_Learning/Unsupervised_Learning/01_Clusteres_enfermedades_cardiovasculares)**

- **Proyecto 2: Influencias en el Nivel de Estrés**  
  - **Descripción:** Este proyecto analiza los factores que influyen en los niveles de estrés de los empleados mediante clustering y análisis exploratorio. Mediante el clustering y Análisis de Componentes Principales (PCA), se identifican las condiciones laborales o factores personales que afectan más al estrés, lo que permite desarrollar intervenciones específicas para reducirlo en el entorno laboral.  
  - **Tecnologías:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
  - **[Ver Proyecto](./Proyectos_Machine_Learning/Unsupervised_Learning/02_Influencias_nivel_estres)**

### 🤖 Proyectos Deep Learning

- **Proyecto 1: Clasificación de Enfermedades Cardiovasculares DL**  
  - **Descripción:** Este proyecto implementa una red neuronal para predecir enfermedades cardiovasculares a partir de múltiples métricas de salud. El modelo se entrena en datos estructurados y se ajusta usando regularización y ajuste de hiperparámetros. Además, se utilizan valores de SHAP para interpretar las decisiones del modelo, mostrando las características que más influyen en sus predicciones.  
  - **Tecnologías:** TensorFlow, Keras, Scikit-Learn, SHAP  
  - **[Ver Proyecto](./Proyectos_Deep_Learning/01_Clasificacion_Enfermedades_Cardiovasculares)**

- **Proyecto 2: Clasificación de Imágenes**  
  - **Descripción:** Este proyecto implementa una red neuronal convolucional (CNN) para clasificar dígitos escritos a mano del conjunto de datos MNIST. Se aplican técnicas de regularización como Dropout y EarlyStopping para evitar el sobreajuste, y se utilizan herramientas de visualización como la Matriz de Confusión para evaluar el rendimiento del modelo.  
  - **Tecnologías:** TensorFlow, Keras, Matplotlib, Seaborn  
  - **[Ver Proyecto](./Proyectos_Deep_Learning/02_Clasificacion_Imagenes)**

- **Proyecto 3: Predicción de Calidad de Vino**  
  - **Descripción:** Utilizando un conjunto de datos con propiedades fisicoquímicas de vinos blancos, este proyecto desarrolla un modelo de regresión de aprendizaje profundo para predecir la calidad del vino. Se emplean técnicas de ingeniería de características y normalización, y se entrena una red neuronal para obtener predicciones precisas.  
  - **Tecnologías:** PyTorch, Numpy, Scikit-learn  
  - **[Ver Proyecto](./Proyectos_Deep_Learning/03_Prediccion_Calidad_Vino)**

- **Proyecto 4: Recomendador de Películas usando Word Embeddings**  
  - **Descripción:** Este proyecto utiliza embeddings de Word2Vec para desarrollar un sistema de recomendación de películas basado en la similitud de sus sinopsis. Se procesan datos de texto con técnicas de procesamiento de lenguaje natural (NLP) y las sinopsis vectorizadas permiten cálculos eficientes de similitud.  
  - **Tecnologías:** spaCy, Gensim, Scikit-learn  
  - **[Ver Proyecto](./Proyectos_Deep_Learning/04_Recomendador_peliculas)**

- **Proyecto 5: Análisis de Sentimientos en Reseñas de Películas**  
  - **Descripción:** Este proyecto utiliza redes LSTM para analizar el sentimiento en reseñas de películas de IMDb. Con técnicas de clasificación y embeddings de palabras, el modelo clasifica reseñas como positivas o negativas. Incluye preprocesamiento de datos, tokenización y visualización de tendencias de sentimiento en las reseñas.  
  - **Tecnologías:** TensorFlow/Keras, NLTK, Gensim  
  - **[Ver Proyecto](./Proyectos_Deep_Learning/05_Analisis_Sentimientos_Reseñas_Peliculas)**
